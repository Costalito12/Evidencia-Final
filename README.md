# Método de Newton-Raphson en Java

Este proyecto implementa el método numérico de **Newton-Raphson** en el lenguaje **Java** para encontrar raíces de funciones reales.

## Descripción

El método de Newton-Raphson es un algoritmo iterativo que aproxima la raíz de una función utilizando derivadas. En este proyecto:

- Se utiliza la función `f(x) = x^2 - 2` como ejemplo.
- Se solicita al usuario: Valor inicial `x0`, Tolerancia deseada, y Máximo de iteraciones
